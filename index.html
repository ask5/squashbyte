Squash Byte

Squash Byte is a fun game that involves squashing 8-bit binary numbers, also known as bytes, using decimal numbers. The game starts with a grid of balls, with each row representing a random byte with a value between 1 and 255. Each ball has a value of either 0 or 1, represented by black or white color respectively.

The objective of the game is to prevent the grid from touching the bottom of the canvas by squashing bytes. To squash a byte, you need to enter the equivalent decimal number using the keypad on the screen or your computer keyboard. For each squashed byte, you earn points equal to its decimal value. The larger the value of the byte, the greater the reward.

Wrong entries are counted as fouls and will cost you 5 points. The speed of the grid increases as you score more points, making the game more challenging. The game ends when the grid touches the bottom of the canvas. Squash Byte is a unique game that combines math skills with quick reflexes and hand-eye coordination.
